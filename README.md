# AI-Powered SRE Incident Analyzer

AI-powered incident analysis and troubleshooting assistant built with Python, LangChain, Ollama, and Streamlit.

## Features
- Upload and analyze operational log files
- AI-generated incident summaries
- Root cause analysis
- Automated remediation recommendations
- Severity classification for production-style incidents
- Local LLM integration using Ollama

## Tech Stack
- Python
- LangChain
- Ollama
- Streamlit
- Llama 3.1 / Qwen2.5
- Prompt Engineering

## Example Use Cases
- Application failure analysis
- Service outage investigation
- API latency troubleshooting
- Kafka consumer diagnostics
- Reliability engineering workflows

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
